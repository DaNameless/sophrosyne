# Sophrosyne

## Project Structure

```
sophrosyne/
├── code/
│   ├── modules/      # Python modules (.py)
│   └── notebooks/    # Jupyter notebooks (.ipynb)
├── outputs/          # Saved figures (git-ignored)
└── README.md
```

## Notes

- The `outputs/` directory is excluded from version control to keep the repository lightweight. Figures can be regenerated by running the notebooks.
